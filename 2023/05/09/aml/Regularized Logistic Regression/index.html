Regularized Logistic Regression

A binary classifier that assigns a label ∈ {−1, 1} to a sample ∈ (a row vector) in terms of a winner-takes-all decision

with parameter (a row vector). We assume that the intercept has been absorbed into by appending a column of ones to the feature matrix X. The posterior probability is modelled by
the sigmoid function

and the optimal parameter vector is found by minimizing the regularized logistic loss (β) over
the training set

where λ is the inverse regularization parameter (i.e. small values of λ lead to strong regularization).
